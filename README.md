# Heco Tap

这是 `heco` (HMOS Ecosystem CLI) 的 Homebrew Tap 仓库，用于在 macOS 环境下分发和安装 `heco`。

`heco` 是深度整合了 `hvigor`、`hdc` 等 DevEco Studio 工具链的命令行工具，方便鸿蒙应用开发者和 AI 调用。

详细信息和源码请访问：[https://github.com/heco-cli/heco](https://github.com/heco-cli/heco)

## 🍏 安装 (macOS)

支持 macOS (Intel / Apple Silicon)。

```bash
# 添加 Tap 仓库
brew tap heco-cli/tap

# 安装 heco
brew install heco
```

**更新 heco：**
```bash
brew upgrade heco
```

---

> **注意：** 
> 1. 上述安装命令中的 `heco-cli/tap` 假定您将此仓库推送到 `https://github.com/heco-cli/homebrew-tap`。
> 2. 当前版本 `heco` 处于快速迭代阶段，请随时保持更新以获取最新特性。
